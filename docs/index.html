Welcome

Here you will find statically compiled FFmpeg and FFprobe binaries for Linux systems. These builds are created using musl toolchains for maximum compatibility.

  • Built with musl libc for better portability
  • Available for x86_64, aarch64, armv7, x86,and riscv64 etc
